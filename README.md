# ShopFast — Repositório de Inspeção | Aula 12

## Objetivo

Este repositório contém código Python com defeitos para praticar **inspeção estática** — análise de código sem executá-lo.

Os alunos devem identificar os defeitos e rastreá-los a uma das 4 categorias do checklist:

1. **Nomenclatura**
2. **Testabilidade**
3. **Segurança**
4. **Tratamento de Erros**

